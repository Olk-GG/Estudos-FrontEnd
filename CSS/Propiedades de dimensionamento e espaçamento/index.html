Propiedades básicas do CSS

Altura e Largura
  • width: largura
  • height: altura
  • initial: valor inicial (default)
  • auto: valor automático
  • inherit: valor herdado do elemento pai
                    
exemplo:

                    
                        .exemplo{
                            width: inherit;
                            height: 200px;
                        }
Max e Min
  • max e min: define o minimo e o maximo de um elemento
  • max-width: largura maxima
  • max-height: altura maxima
  • min-width: largura minima
  • min-height: altura minima
                    
exemplo:

                    
                        .exemplo{
                            max-width: 600px;
                            max-height: 600px;
                            min-width: 200px;
                            min-height: 200px;
                        }
Margin
  • margin: Valor aplicado a todos os lados
    • Dois valores: Aplica cima e baixo
    • Três valores: Aplica cima, lado e baixo
    • Quatro valores: Aplica cima, direita, baixo e esquerda
  • margin-top: margem superior
  • margin-left: margem esquerda
  • margin-right: margem direita
  • margin-bottom: margem inferior
                    
exemplo:

                    
                        .exemplo{
                            margin: 10px, 10px, 10px, 10px;
                            margin-top: 20px;
                            margin-right: 30px;
                            margin-bottom: 40px;
                            margin-left: 50px;
                        }

                        
// utilizando auto pode-se centralizar seu elemento: 

                        margin: 10px auto;
Padding

O padding assim como margin, pode ser aplicado a todos os lados, o difetencial dos dois é:

padding separa os elementos inernos;
margin separa os elementos externos.

  • padding: Valor aplicado a todos os lados
    • Dois valores: Aplica cima e baixo
    • Três valores: Aplica cima, lado e baixo
    • Quatro valores: Aplica cima, direita, baixo e esquerda
  • padding-top: padding superior
  • padding-left: padding esquerda
Box sizing

Quando falamos de box sizing, estamos falando de como o navegador vai renderizar nosso elemento. Para isso devemos ficar atento ao tamanho que atribuimos ao elemento.

Pois o tamnho do elemento é calculado da seguinte forma:
tamanho = valor (height x width) + border + padding

  • box-sizing: Define como o navegador vai redimensionar o elemento
  • content-box: Ele soma os valores definidos, como no exemplo a cima
  • border-box: Redimensiona o padding e o border para se adequar ao height e width definidos.
                    
                        
exemplo

                        .exemplo{
                            box-sizing: border-box;
                            box-sizing: content-box;
                        }
