Seletores CSS

  • Seletor por tipo: Busaca elementos por uma tag HTML especifica
  • Seletor por ID (#): Busca elementos através do atributo "id"
  • Seletor por Classe (.): Busca elementos através do atributo "class"
  • Seletor Universal: (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscas atributos com um valor especifico.
                    
Exemplos:

                    
                    
Seletor por tipo:

                    h1 {
                        color:red
                        background-color:#f1f1f1;
                    }
                    

                    
Seletor por ID:

                    #titulo-pincipal {
                        color:red
                        background-color:#f1f1f1;
                    }
                    

                    
Seletor por class:

                    .tabela-horas {
                        color:red
                        background-color:#f1f1f1;
                    }
                    

                    
Seletor Universal:

                    * {
                        color:red
                        background-color:#f1f1f1;
                    }
                    

                    
Seletor de Atributo:

                    [title] {
                        color:red;
                        background-color:#f1f1f1;
                    }

                    ou

                    [title="seletor-atributo"] {
                        color:red;
                        background-color:#f1f1f1;
                    }

Agrupando Seletores

Podemos aplicas as mesmas regras CSS para multiplos seletores diferentes. Para isso, podemon separar os seletores utilizando uma virgula (,).

                
Exemplo:

                
                    .agrupando-seletores h1, .agrupando-seletores p {
                        color:blueviolet
                    }

Combinador Descendente

Para selecionar elementos descendentes, podemos utilizar as seguintes relações entre dois seletores:

  1. Combinador Descendente (espaço)
  2. Combinado filho (>)
  3. Combinador irmão adjacente (+)
  4. Combinador irmão em geral (~)

Ficaria:

[SELETOR][COMBINADOR][SELETOR]

Exemplo:

  • Item 1
    • Subitem A
    • Subitem B
  • Item 2
    • Subitem A
    • Subitem B
  • Item 3
    • Subitem A
    • Subitem B
