Formatando Textos HTML

Para a formatação de textos html, usa-se diversas tags, sendo as mais comuns:

  • <b>: Torna seu texto em negrito;
  • <strong>: Torna o texto em negrito, porém é utilizado quando se quer dar mais enfase em partes do texto (visualmente é igual a tag <b>);
  • <i>: Essa tag torna pate do texto em italico;
  • <mark>: Marca parte do texto;
  • <u>: Sublinha parte do texto;
  • <s>: Grifa parte do texto;
  • <p>: Cria novos paragrafos;
  • <blockquote>: Usada comumente para citações;
  • <sup>: Faz com que apareça meio caractere acima da linha normal exemplo;
  • <sub>: Faz com que apareça meio caractere abaixo da linha normal exemplo;
  • <hx>: Cria títulos ;
  • <small>: Deixa parte do texto pequeno;

Estrutura HTMl

Tag <div>

A tag div é utilizada para definir seções e divisões dentro de um pagina html, ou seja ela é um container que podem ser estilizados com CSS ou manipulados com JavaScript.

Tag span

A tag <span> é um contêiner embutido usado para marcar uma parte de um texto ou uma parte de um documento, pode ser facilmente estilizada por CSS ou manipulada com JavaScript usando o atributo class ou id.

Tag fieldset

A tag é usada para agrupar elementos relacionados em um formulário. Ela desenha uma caixa ao redor dos elementos relacionados.

Juntamente com a tag fieldset, usa-se a tag <legend> para se adicionar um título.

Tag iframe

A tag iframe é utilizada para criar um quadro embutido, comumente usado para incorporar outro documento dentro do documento HTML atual. Como janelas que apresentão conteude de sites externos.
